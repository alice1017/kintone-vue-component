> `k-spinner` コンポーネントはローディング中にユーザーが画面を触れないようにするために使用されます。

## デモ

![]({{ "/images/k-spinner.png" | relative_url }})

## 使い方

```vue
<template>
  <k-spinner :visible="true" />
</template>
```

## Attreibute

| Name    | Description                                       | Type    | Required |
| ------- | ------------------------------------------------- | ------- | -------- |
| visible | スピナーを表示するかどうか.<br>デフォルト値: true | Boolean | No       |

## Event

なし
